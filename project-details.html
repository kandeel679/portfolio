Full Stack Face Recognition App

Completed in February 2024

Overview

A full-stack React application featuring face recognition capabilities and a robust login system. The app integrates with a secure database and leverages the Clarifai API for advanced image processing and face detection.

Key Features

  • User authentication system with secure password hashing
  • Real-time face detection in uploaded images
  • PostgreSQL database integration for user data management
  • RESTful API endpoints for seamless frontend-backend communication
  • Responsive design for optimal viewing across devices

Implementation Details

The application is built with a React frontend that communicates with a Node.js/Express backend. User data is stored securely in a PostgreSQL database with passwords hashed using BCrypt. The face recognition feature utilizes the Clarifai API's machine learning models for accurate detection. JWT tokens are implemented for secure user sessions.

Technical Stack

React.js Node.js Express PostgreSQL Clarifai API JWT BCrypt
